### PR #5: TypeScript 指令类型定义（月栖白）

**目标**: 写出合并版 `commands.ts`，让芝士番薯立刻开工渲染引擎

**时间**: 10-15 分钟

**文件**:
```
frontend/src/types/commands.ts   ← 合并版指令 TS 类型（全部）
```

**内容**: 包含双方所有指令类型
- 8 种图形指令（芝士番薯）：circle/rect/line/curve/polygon/ellipse/arc/text
- 对象标识字段（月栖白）：id/label/role/groupId/cellId
- 对象编辑指令（月栖白）：update_object/move_object/delete_object
- 控制指令（合并）：setColor/setWidth/setBrush/clear/undo/speak/wait
- 画笔/渐变类型（芝士番薯）：fillGradient/mode/brush/duration
- API 请求/响应类型

**验收**: `npx tsc --noEmit` 通过，芝士番薯可以 `import { DrawInstruction }` 开始写 renderer
