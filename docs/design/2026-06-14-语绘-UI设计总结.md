# 语绘 (Voice Canvas) — UI 设计总结

> 2026-06-14 · 芝士番薯

---

## 一、页面结构

### 1.1 两页设计

| 页面 | 组件 | 说明 |
|------|------|------|
| **登录页** | `LoginPage.tsx` | 品牌展示 + 用户名字输入 + 进入画室 |
| **绘画页** | `App.tsx` | 画布 + 工具栏 + 语音控制 |

路由方式：React state `page: 'login' | 'draw'`，0.6s 过渡动画。

---

## 二、登录页

### 2.1 视觉设计

- **背景**：暖宣纸色 `#faf8f5` + 5 团水彩晕染色块（暖灰棕系径向渐变，12 秒呼吸动画）
- **卡片**：白色玻璃卡片，居中放置
- **海浪**：三层 SVG 波浪在底部浮动（远/中/近，6-10 秒推拉动画）
- **装饰**：5 个浪花泡沫圆点漂浮

### 2.2 品牌标识

- **🎨 图标**：48px，带 `drop-shadow` 蓝调投影，3 秒上下浮动
- **「语绘」标题**：Google Fonts Zhi Mang Xing（智芒行楷），72px，墨黑渐变 `#2c→#1a→#33`，微投影模拟墨迹渗透
- **Voice Canvas**：12px，同字体，灰色，宽字距

### 2.3 输入区

- **文字框**：白色圆角，focus 蓝光边框 + 阴影
- **按钮**：「进入画室」+ 4 根声纹跳动条，黑灰渐变背景，hover 上浮 + 阴影

### 2.4 动画清单

| 动画 | 元素 | 时长 |
|------|------|------|
| `watercolor-shift` | 背景水彩晕染 | 12s |
| `ocean-swell` | 底部海浪 SVG | 6-10s |
| `dot-float` | 浪花泡沫圆点 | 4s |
| `icon-float` | 🎨 图标 | 3s |
| `btn-wbar-dance` | 按钮声纹条 | 0.5s |
| `card-in` | 卡片入场 | 0.8s |

## 三、绘画页

### 3.1 布局结构

```
┌──────────────────────────────────────┐
│  ← 语绘 VOICE CANVAS      [用户名]   │  Header
├──────────────────────────────────────┤
│  画笔 ✏️彩铅 🖌️颜料 │ 颜色 🔴🟠...⚪  │  PaletteBar
├──────────────────────────────────────┤
│  🟢 状态 [🎤录音] [↩] [↪] [🗑]      │  VoiceBar
├──────────────────────────────────────┤
│  [指令历史标签...]                    │  CommandHistory
│  [调试输入...              ] [发送]  │  Debug Row
├──────────────────────────────────────┤
│                                      │
│           Canvas 画布 (900×550)      │
│                                      │
└──────────────────────────────────────┘
```

### 3.2 Header

- **← 返回按钮**：点击回到登录页
- **「语绘」**：Zhi Mang Xing 行楷，36px，墨黑渐变
- **VOICE CANVAS**：同字体，12px，灰色
- **用户名标签**：右侧圆角灰底

### 3.3 PaletteBar 面板

- **画笔区**：✏️ 彩铅（排线填充）、🖌️ 颜料笔（笔触纹理）
- **颜色区**：12 色圆点 + 中文名（红/橙/黄/绿/青/蓝/紫/粉/棕/灰/黑/白）
- 横排卡片式，header 下方

### 3.4 VoiceBar 控制栏

- **状态指示点**：6 色（绿/蓝/橙/紫/粉/红）+ 脉冲/呼吸动画
- **录音按钮**：圆角胶囊，录音中红色发光 + 扩散光环
- **工具按钮**：撤销 ↩ / 重做 ↪ / 清空 🗑，圆角图标

### 3.5 状态动画

| 状态 | 视觉效果 |
|------|----------|
| idle | 绿色圆点常亮 |
| recording | 蓝色圆点 `pulse-ring` 脉冲 + 按钮红色发光 |
| transcribing | 橙色圆点 `dot-pulse` 呼吸 |
| generating | 紫色圆点 `dot-pulse` 呼吸 |
| drawing | 粉色圆点 `dot-pulse` 呼吸 |
| error | 红色圆点常亮 |
